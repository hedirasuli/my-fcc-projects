# TypeScript Task Management System (Mini-Project)

A functional project to demonstrate advanced TypeScript concepts.

## Features Implemented:
- **Generics**: Used for a reusable Repository pattern.
- **Advanced Types**: Leveraging Enums, Interfaces, and Tuples.
- **Generic Constraints**: Enforcing type safety with `extends` and `keyof`.
- **OOP**: Class inheritance and encapsulation.

## How to run:
```bash
npx ts-node app.ts
